# Markdown Calc

Markdown calc is the idea to treat markdown tables like an excel workbook.

## Example

Input

| Calc Table |        |
| ---------- | -----: |
| Line 1     |   1000 |
| Line 2     |    200 |
| Total      | =B1+B2 |

Output

| Calc Table |      |
| ---------- | ---: |
| Line 1     | 1000 |
| Line 2     |  200 |
| Total      | 1200 |
